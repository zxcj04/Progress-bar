# Simple Library for Progress Bar

## Install
- With pipenv
    - `pipenv install mortalprogressbar`

## Usage

### Import
`from progress import ProgressBar`

### Use

```python
with ProgressBar(total=100, width=30, prefix="Processing...") as bar:
    bar.update(add=1)
```

### Example

```python
with ProgressBar(total=100, width=30, prefix="Processing...") as bar:
    for _ in range(total):
        bar.update(add=1)
        time.sleep(0.05)
```
